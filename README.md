1. Student Mark Analyzer
A script to process academic data for a small group of students.
Features:
Calculates individual student averages.
Identifies the top-performing student (Topper).
Computes averages for specific subjects (Math, Science, English).
Concepts Used: Dictionaries, list indexing, and loops.

2. Contact Book Program
A CLI-based (Command Line Interface) application to manage a personal directory.
Features:
Add: Save new names and phone numbers.
View: Display all saved contacts in a clean format.
Search/Update: Quickly find or modify existing details.
Delete: Remove unwanted entries.
Concepts Used: CRUD operations, input sanitization (.strip(), .lower()), and infinite loops with exit conditions.

3. Simple Banking System
A simulation of a banking interface for account management.
Features:
Account creation with an initial balance.
Real-time balance inquiries.
Deposit and withdrawal functionality with insufficient funds validation.
Concepts Used: Global state management (dictionary) and conditional logic.
